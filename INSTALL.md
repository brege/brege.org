## For a whole new project directory or new machine

1. `git clone git@github.com:brege/brege.org.git`

2. `cd brege.org`

## Get the theme.  We are now using [papermod](https://github.com/adityatelange/hugo-PaperMod)

3. `mkdir -p themes/hugo-papermod`

4. `git clone git@github.com:adityatelange/hugo-PaperMod.git themes/hugo-papermod/`

5. check: `hugo server`

Should probably use modules, but this works for now.
